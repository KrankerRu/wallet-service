# Wallet Service

Сервис для управления балансом кошелька с поддержкой конкурентных операций.

## Что умеет

- Пополнять и списывать деньги с кошелька
- Показывать текущий баланс
- Работать под высокой нагрузкой (1000 RPS на один кошелек)
- Красиво падать с понятными ошибками

## Технологии

- Java 17
- Spring Boot 3
- PostgreSQL
- Liquibase (миграции)
- Docker / docker-compose
- JUnit 5 + Mockito

## Как запустить

### Через Docker (рекомендуется)

# Собрать jar
mvn clean package -DskipTests

# Поднять контейнеры
docker-compose up -d --build

Сервис будет доступен на http://localhost:8080
